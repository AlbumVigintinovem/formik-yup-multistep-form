
## Form Application
Multistep form with Formik and Yup.
## Run Locally

Clone the project

```bash
  git clone https://github.com/AlbumVigintinovem/e-commerce-store.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the project

```bash
  npm start
```

**Client:** React, Bootstrap
